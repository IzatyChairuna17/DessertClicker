Deskripsi Aplikasi

Dessert Clicker adalah aplikasi sederhana di mana pengguna dapat:

1. Menekan gambar dessert untuk menjual dessert.

2. Melihat jumlah dessert yang telah terjual.

3. Melihat total pendapatan yang diperoleh.

4. Dessert akan berubah otomatis sesuai jumlah penjualan (dessert level).
